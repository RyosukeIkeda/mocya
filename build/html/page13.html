食パングラタン

2017/04/16

こんにちは、もちゃです。

今日は、ノーマル食パンを使ってパングラタンを作ってみようと思います。

ノーマル食パンのレシピはこちら

材料
  • ノーマル食パン…1斤
【もちゃ家のホワイトソース】
  • バター…10g
  • 小麦粉…大さじ4
  • 牛乳…600ml
  • ベーコン…お好みの量(もちゃは70g入れました)
  • 玉ねぎ…お好みの量(もちゃは1個入れました)
  • 固形コンソメ…2個
  • とろけるチーズ…適量
作り方

  1. まず、ホワイトソースを作ります。

    バターをフライパンに入れ弱火にかけ、溶けてきたところでベーコンを加える。

  2. ベーコンから脂が出てきたら玉ねぎを入れ、玉ねぎが透明になるまで炒める。

  3. 玉ねぎが透明になったら火を消し、小麦粉を入れまんべんなく混ぜ合わせる。

    ↑このくらい混ぜ合わせればOK

  4. 牛乳、コンソメを入れ弱火で煮詰めていく。とろみが出たらホワイトソースの完成。

  5. ノーマル食パンを写真のようにくり抜き、器のようにします。

  6. くり抜いた部分に先程作ったホワイトソースをお好みの量まで入れ、とろけるチーズをかけてオーブンでお好みの焼き色になるまで焼いたら完成！！

    (もちゃは190度で余熱なし5分焼きました)

チーズとろりで美味しそう！！

※オーブンから取り出す時は、熱いうえにずっしりと重いので火傷に気をつけて下さいね。

ホワイトソースがとんろり！！！

食パンにソースが染みてまろやかなうえにチーズの香りで美味しすぎる！！

これは是非焼きたてを味わって頂きたいです！

もちゃ夫と美味しい美味しいと、ハフハフしながら食べました！

普段朝ご飯は食パン1枚ずつのもちゃ夫婦ですが、このパングラタンは一気に3/4食べてしまいました。

見栄えしますが、焼き上がりまで大体30分くらいで出来るので、休日の朝ご飯におすすめです☆

※もちゃ家のホワイトソースは、もちゃがこってり系が苦手なのでかなりあっさりめです。 バターも少なければ生クリームも入れず牛乳のみで作るので物足りないぜ！って方はバターの増量と牛乳の半量を生クリームに置き換えてみて下さい。
